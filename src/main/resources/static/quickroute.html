  • Quick Route

A multi-stop trip planner written in Java using Spring Web MVC that takes an input of locations and generates the most efficient route to visit all the given locations.

  • Github
  • Web Application
  • Stack: Java, HTML, CSS
  • Frameworks: Spring Web MVC

Features:

  • Takes a user input of locations and generates the most efficient route to visit all the given locations.
  • The starting location can be chosen by the user or chosen to optimise efficiency.
  • Calculates route based off either duration or distance.
  • Duration/distance values account for different transport methods (Driving, Transit, Walking) as specified by the user.

How it works:

The Google Maps Api was used to get the data for the duration/distance between locations and a complete weighted digraph was created using those values as weights and the locations as nodes.
A recursion based graph traversal algorithm utilises dynamic programming to find the Shortest Hamiltonian Path in the digraph.
