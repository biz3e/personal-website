  • A Maze Game

A randomly generated maze game with 20 difficulty levels written in GDScript and utilises the Godot game engine.

  • Github
  • Stack: GDScript
  • Game Engine: Godot

Features:

  • Has 20 difficulty levels, where each increase in level increases the number of cells in the maze.
  • The maze is randomly generate each time a level is chosen.
  • Multiple background music choices.

How it works:

The maze is divided into a specified number of cells based off the chosen difficulty levels and a hunt and kill algorithm is used to randomly generate the maze.

The algorithm: From the starting cell, the algorithm randomly choses a direction to travel and travels a one cell distance in that direction.
Each time the algorithm travels a cell, it marks the current cell as visited and randomly chooses another direction to travel.
If there are no more unvisited cells adjacent to the current cell, the algorithm chooses a visited cell that is adjacent to at least 1 unvisited cell and starts randomly choosing a direction to travel again.
